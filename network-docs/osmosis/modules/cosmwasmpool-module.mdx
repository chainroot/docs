The CosmWasmPool module in Osmosis bridges CosmWasm smart contracts with liquidity pool functionalities.

## Key Functionalities
- **Smart Contract Integration**: Enables the use of CosmWasm smart contracts within liquidity pools.
- **Enhanced Pool Features**: Provides additional functionalities for liquidity pools through smart contracts.
