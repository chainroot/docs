# Network Modules of Osmosis

Osmosis features several key modules that together create a robust and versatile DeFi platform.

## Overview of Modules

- **Liquidity Module**: Manages the creation and operation of liquidity pools.
- **Swap Module**: Facilitates asset swaps within and across different blockchains.
- **Governance Module**: Enables community-driven governance and decision-making.

## Module Functionalities

Each module in Osmosis plays a crucial role in ensuring the platform's efficiency, security, and user-friendliness.

For a comprehensive understanding of the modules in Osmosis, visit [Osmosis Network](https://osmosis.zone/).
